Tools Used

1.Keybr.com

A web-based typing tutor that reinforces touch typing muscle memory through repetitive practice. Used after we
graduated from Typing.com.

2.Typing.com

A comprehensive typing platform used to strengthen and build a foundation in touch typing muscle memory. This website was
chosen for its extensive range of lessons and practice exercises as it eases you into touch typing row by row.

3.OBS Studio

A free and open-source software for video recording and live streaming. Used for recording practice sessions and
creating videos as proof of progress.

4. Google Docs

A web-based document editor that allows for real-time collaboration and editing. Used for planning the initial project
outline and documentation.

5. Visual Studio Code

A source-code editor developed by Microsoft for Windows, Linux, and macOS. Our go-to editor for writing and editing code.
allows us to collaborate seamlessly using the Live Share extension

6. Github

A web-based platform used for version control and collaborative software development. Reliable website to store our code.
